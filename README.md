# MoodTune

## Project Description
MoodTune is an application that recommends music based on your mood. By answering a few questions, the app detects your mood and suggests a playlist tailored to your emotional state.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

## Features
1. Mood detection through a questionnaire.
2. Personalized music recommendations.
3. User-friendly interface.
4. Responsive design for mobile and desktop.

## Installation
```bash
# Clone the repository
git clone https://github.com/your-repo/MoodTune.git

# Navigate to the backend directory
cd backend

# Install dependencies
npm install
```

## Run Commands
```bash
# Start the backend server
node server.js
```
